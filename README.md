# SafePay Banking Dashboard

SafePay is a modern, clean, and professional banking web application UI. It features a comprehensive dashboard, fund transfer capabilities, and transaction history management.

## Features

- **Professional Dashboard**: Overview of balance, income, expenses, and savings.
- **Secure Fund Transfers**: Intuitive transfer form with account validation and OTP simulation.
- **Transaction History**: Detailed tracking of all financial activities with filtering and search.
- **Responsive Design**: Fully optimized for both desktop and mobile devices.
- **Modern UI/UX**: Built with Tailwind CSS and Motion for a premium feel.

## Tech Stack

- **Frontend**: React 19
- **Styling**: Tailwind CSS 4
- **Animations**: Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/safepay-dashboard.git
   cd safepay-dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the project for production.
- `npm run preview`: Previews the production build locally.
- `npm run lint`: Runs TypeScript type checking.
